Image Effects #back to top

In this template, We have used a Spotify Inspired Duotone Effect for images. So if you are replacing the images and need the same Duotone effects, Please follow the steps.

Duotone Action #back to top

Duotone Photoshop Action file is already included with the template, Just head over to assets/ folder and find the *.atn file.

Further Steps:

  1. Install PS Action by Loading the *.atn file from Photoshop
  2. Open the image to create the effect
  3. From the "Actions" Panel, Choose "Spotify Duotone Effect" and click run (Play button)
  4. Now, new layers will be created, Expand the Layer Group to see color options
  5. To change the effect color map, please hide and show the layers and choose the one you like
  6. Done. Adjust the Brightness and the Opacity as you like and Save it.

The heart of a Music & Band Website is "Music". So we have already included different ways to include Audio and Video in the different part of this website. Please read below to know more.

Fixed Audio Player #back to top

A Responsive HTML5 Audio player is placed in the bottom of the website which will help users to play music while they surf website. This Audio Player uses "jPlayer" jQuery Plugin which enable lot of customization options. If something is not mentioned here, You may always chech the original jPlayer Documentation here

jPlayer is initiated in /js/script.js file (FROM LINE 145) . You may adjust the settings by editing the code there.

Now edit the script.js file to change the Path of existind dummy songs to your original songs. The code will look like below.

Autoplay

You can also adjust the Autoplay settings from the JS code, Currently its set to false. You may enable it by changing the value to true. It is recommended that you keep the autoplay set to false as autoplay might affect user experience as it distracts users.

    autoPlay: true 
 }

Album Play Button #back to top

Album play button which is shown in the Home Page as well as Album Single Page is currently linked to the jPlayer Playlist. So if you want to change the Song, Just change the data-playlist-id value in the HTML file. The count starts from 0,1,2 etc.. 

                        <div class="audio-play-btn" data-playlist-id="0">
                        <!-- Currently "0" plays "Cro Magnon Man" as its the first one in the playlist. -->

Redirect on Success (Thank you page)

If you want your users to be redirected to another page after completing the form successfully such as Thank you page or Success page. It is really easy to do with this Template. Open index.html and Just add data-redirect="/path/to/thanks.html" to <form> element and you are done. See example: 

<!-- // NOTE THE data-redirect attribute -->
 <form action="php/contact.php" method="POST" data-redirect="success.html">

Alternatively, If you don't want to redirect your users and want to stay in same page, Just change the data-redirect attribute value to none.

Countdown Timer #back to top

jQuery countdown timer can be seen in event-countdown.html file. To change the time, Open HTML and find the data-event-date attribute. Then replace the value with your event date. See example below

                        <!-- Change the date & time in same format -->
                         <ul id="countdown" data-event-date="12 february 2018 20:30:00">

For further settings, you can always head over to js/script.js to customize it.

Eventbrite Popup #back to top

If you are hosting a free or paid event, it will be a great idea to let the users book their slots. We have included an option to book tickets via Eventbrite platform. You can see it live in event-countdown.html page by clicking "Buy Tickets" option. To customize the Eventbrite popup, please follow the steps below.

  1. Login to Eventbrite
  2. Click My Events and Selet your event
  3. From Event Dashboard Sidebar, Click Widgets under Invite and Promote
  4. Select Ticket form and copy the HTML code
  5. Now copy only the iframe src URL
  6. And replace the URL in HTMLiframe section

Example:

<!-- See the "src" value below. change that to your URL -->
<iframe  src="//eventbrite.com/tickets-external?eid=9293674631&ref=etckt" frameborder="0" height="350" width="100%" ...></iframe>

Embed Tweet #back to top

There's an embed tweet section in the fan page. This is one of the best way to show real world tweets about your band. Using real tweets will show your customer likings and thus increase your trust. To change the tweet, please follow the steps

  1. Open Twitter and find the tweet you'd like to embed
  2. In the first line, you will find the Posted Time eg: 2h, May 14 etc..
  3. Right click on the Time link and copy the Link Address
  4. You may also Open the tweet to find the Post URL
  5. The link will look like https://twitter.com/user/status/123456
  6. Now paste the link in the HTML Fan Page section by replacing the existing URL

Example:

                        <!-- Replace the href value -->
                        <a href="https://twitter.com/katyperry/status/691023759667400704"></a>

Website Optimization Tips #back to top

A Fast & Optimized Website has several factors which needs to be implemented in order to achieve the desired results. There are several Optimization Techniques available which will definitely affect your Website's Performance in a Positive Way & we want to share a few of them with you:

  1. gZip Compression & Browser Caching

    This is probably one of the Most Important Techniques you should definitely implement in order to bump up your Website's Loading Speed. gZip Compression is used to compress the Files that are delivered when loading a Website. It covers HTML, CSS, Javascript & Font Files along with other miscellaneous text files. Where as Browser Caching also covers Images & Videos apart from including the above files. This is used to saves the Static Data in your Browser itself so that when you open the Next Pages on the Same Website, the content does not gets Downloaded again, loading the Website fast.

    gZip Compression & Browser Caching can be enabled using the .htaccess File on an Apache Web Server. You can use the Codes from here: https://github.com/h5bp/html5-boilerplate/blob/master/dist/.htaccess to enable these modules on your server.

  2. Image Compression & Optimization

    We tend to use Lots of Images on our Websites but we often do not make efforts to Compress & Optimize them. Remember, the Larger the Image, the more time it takes to download and therefore this slows your website loading times affecting User Experience. Your customer will leave your website if it does not load within 3-5 Seconds which adversely affects your Sales. Therefore, it is important to Resize, Optimize & Compress your Images before using it on your Website. Here are some Tips which might come handy in optimizing images:

    1. Resize your Images: Resize your Images before using it on your Website. Do not just Download an Image & place it as it is in your Website's <img> Tag without resizing it. The size/resolution of the Image matters since it is not recommended to use an Image size of 1200px x 800px in a Content Size of 300px x 200px as this is unnecessary. Resize it to 300px x 200px
    2. Image Formats: There are three common file types that are used for web images which are JPEG, GIF, & PNG. For images with a Flat Background use JPEG images, for images with a Transparent background use PNG images and for images with Animations use GIF images.
    3. Compressing Images: Images Compression is important as it considerably reduces the size without losing the quality. There are several FREE Image Optimization Tools available to Download.
      For MAC use ImageOptim
      For Windows use Riot for compressing JPEG Images & PNG Gauntlet for PNG Images.

  3. CSS & jQuery Minifications

    It is also recommended that you Combine & Minify all your CSS Files to a single CSS File & all Javascript Files to a single JS File since Minification reduces the size of the File and Combining the files helps in reducing the number of HTTP requests made to the server. This is also an Important Factor in increasing the speed of your website. There are several tools available online to Minify your CSS & JS Files. Our recommendations are:
    For CSS use CSS Minifier and For Javascript use Javascript Minifier.

  4. Content Delivery Network

    You can use a CDN to further speed up your website. You can use the CDN to deliver static files of your website like CSS, JS, Images & Font Files. There are several CDN Hosting Providers available on the Internet but we would recommend MaxCDN or CloudFlare. Note: CDN setup requires Extra monthly Fees to setup, so it is completely optional & according to your needs. Cloudflare also has a Free plan, you can try that for free.

  5. Fast Web Hosting Servers

    A lot depends on your Web Hosting Servers, so it is recommended that you choose a Hosting Company/Server that provides a Reliable & a Fast Hosting Service. You can also choose our shared hosting plans. See above.
